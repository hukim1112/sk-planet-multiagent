2. 실습 환경 세팅

환경: Github Codespaces (Github 계정 필수)

📌 초기 설정 절차

  1. Github Codespaces 접속
  2. 우상단 프로필 Settings 좌측 Codespaces 선택
  3. Default idle timeout을 240min으로 설정
  4. Jupyter Notebook 템플릿을 선택하여 클라우드 환경 접속
  5. 접속 후 불필요한 파일 삭제 (루트 경로의 requirements.txt 등)
  6. 터미널에서 아래 명령어 입력하여 repository 복사:
    git clone https://github.com/hukim1112/llmops_class

🚀 다음 절차

  • 복사한 레포지터리 내 README.md를 참고하여 환경 세팅 및 프로젝트 구조/설치 방법 확인
  • .env 파일을 프로젝트 경로에 추가하고, 경로 및 API Key 설정
    ※ API 키는 좌측 문서 탭에서 확인하실 수 있습니다.
